Gerador de Senhas Visuais em C# 

Este repositório contém um aplicativo desktop para geração de senhas seguras, 

desenvolvido como exercício prático de programação durante o curso no SENAI. 

🎯 Objetivo do Projeto

O objetivo principal deste projeto foi praticar a lógica de programação baseada em eventos no Windows Forms , 

a criação de interfaces de usuário e o uso eficiente de classes como Random e StringBuilder para a concatenação de textos dentro de laços de repetição.

✨ Funcionalidades

Tamanho Personalizável: O usuário pode definir o comprimento exato da senha através de um seletor numérico (NumericUpDown).

Geração Aleatória: O sistema sorteia caracteres de forma aleatória para garantir a imprevisibilidade da senha.

Filtros Personalizados (Desafio Implementado): Inclusão de caixas de seleção (CheckBox) que permitem ao usuário escolher se deseja incluir números ou símbolos especiais na senha gerada.

Cópia Rápida (Desafio Implementado): Um botão dedicado que copia automaticamente a senha gerada para a área de transferência do sistema, agilizando o uso.

Modo de Exibição Seguro: A caixa de texto que exibe a senha gerada (TextBox) está configurada apenas para leitura, impedindo que o usuário altere o resultado acidentalmente.

🛠️ Tecnologias e Ferramentas Utilizadas

Linguagem: C# Framework: Aplicativo do Windows Forms (.NET) 

IDE: Visual Studio (2022 ou superior recomendado) para ambiente Windows 

Controle de Versão: Git / GitHub

🚀 Como Executar o ProjetoCaso queira testar a aplicação localmente, 

siga as etapas abaixo:

É necessário ter o Visual Studio instalado na máquina.

Faça o clone deste repositório para o seu computador.

Abra o arquivo do projeto no Visual Studio.

Para compilar e rodar, pressione a tecla F5 ou clique no botão verde "Iniciar" localizado no menu superior.

Na janela que se abrirá, escolha as configurações desejadas e clique no botão para gerar sua senha!
